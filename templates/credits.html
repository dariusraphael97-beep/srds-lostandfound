Statement of Original Work

All application code — including the Flask backend (app.py), HTML templates, CSS stylesheet (style.css), and JavaScript (main.js) — was written entirely from scratch as original work for the FBLA Website Coding & Development competition, 2025–2026.

No website templates, Wix, Squarespace, WordPress, or third-party page builders were used. No pre-built UI component libraries (Bootstrap, Tailwind, Material UI, etc.) were used. All design, layout, animations, and interactivity were coded by hand.

The Smart Match scoring algorithm, Private Proof claim system, Campus Heatmap visualization, and Item Journey Timeline are all original features designed and implemented for this project.
